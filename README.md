# StudySuite

Fast student utilities built with Next.js + Tailwind + shadcn/ui.

## Features
- GPA / Percentage converter (marks or GPA scale)
- Routine planner (weekly grid + local storage)
- Expense splitter (balances + settlement summary)
- Notes to PDF export

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS + shadcn/ui

## Getting Started

Install dependencies and run the dev server:

```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser.

## Scripts
- `npm run dev` - start dev server
- `npm run build` - production build
- `npm run start` - run production server
- `npm run lint` - lint
